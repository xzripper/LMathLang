# LMathLang - Next generation of <a href="https://github.com/xzripper/MathLang/">MathLang</a>.
### Improved version of language, and ported on Lua.

## What's the difference?
#### MathLang.
 - ✔️ **Comments.**
 - ✔️ **Arithmetic.**
 - ✔️ **Algebra.**
 - ✔️ **Easy.**
 - ❌ **Variables.**
 - ❌ **Nice exceptions.**
 - ❌ **Writing text.**
 - ❌ **Instantly run.**
 - ❌ **Nice API.**
 - ❌ **Add-Remove Algebra.**
 - ❌ **Got rid of the semicolons.**
 - ❌ **Plans on future.**
 - ❌ **Nice documentation.**
